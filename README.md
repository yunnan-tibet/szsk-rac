## 组件类型
### 较常用型
#### Form
表单配置化，不用写那么繁杂的form
#### Search
search表单配置化
#### SearchTable（暂未开发）
search + table的组合，不需要考虑分页等逻辑
#### ContentWrapper
通用容器卡片

### 自适应型
#### AutoSizeList
自适应容器下用于列表自适应排列
#### StickyTabs
tabs瀑布展示模块吸顶，整体模块head + tabs吸顶，tab切换与滚动联动
#### ContainImage
图片自适应块居中展示
#### ContainVideo
video自适应块居中展示，包含视频截图功能
<!-- #### ContainAttach

### 图片型
#### ContainImage
图片自适应块居中展示
#### CropPicModal
图片裁剪
#### PictrueJigsaw
一个容器内多图片拼图
